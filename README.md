Consultorio Dental:
Bienvenidos al repositorio oficial del consultorio dental Este proyecto consiste en el desarrollo de una plataforma web dinámica diseñada para agilizar el proceso de consulta y agendamiento de citas dentales, ofreciendo una experiencia de usuario moderna, profesional y eficiente.

Sobre el Proyecto
Nuestra misión fue transformar la gestión de citas tradicional en un proceso digital fluido. El sitio permite a los pacientes:
•	Conocer las especialidades dentales disponibles.
•	Consultar el perfil y la experiencia de nuestro equipo de especialistas.
•	Agendar citas a través de un formulario inteligente con validación en tiempo real.

Tecnologías Usadas
•	HTML5: Estructura semántica de la aplicación.
•	CSS3: Diseño responsivo, efectos de vidrio (Glassmorphism), animaciones y sistema de partículas.
•	JavaScript: Lógica de validación, manejo de ventanas modales y dinamismo del DOM.
•	GitHub: Control de versiones, flujo de trabajo basado en ramas (branches) y gestión de Pull Requests.


Instrucciones de Uso
Clonar el repositorio
Para obtener una copia local del proyecto, ejecuta el siguiente comando en tu terminal:
Bash
git clone https://github.com/tu-usuario/nombre-del-repositorio.git

Ejecutar la aplicación
No se requieren dependencias externas Una vez clonado:
1.	Navega a la carpeta del proyecto.
2.	Abre el archivo index.html en cualquier navegador web moderno (Chrome, Firefox, Edge).

   
 Características Destacadas
•	Fondo de Partículas Animado: Un sistema visual que da vida al sitio mediante puntos flotantes creados matemáticamente para no afectar el rendimiento.
•	Navegación Inteligente: Un Header "sticky" con efecto de desenfoque que acompaña al usuario en todo momento.
•	Tarjetas Interactivas: Información de doctores que se revela suavemente al pasar el cursor (hover), optimizando el espacio visual.
•	Validación de Datos: Formulario que verifica campos obligatorios y formato de teléfono antes de confirmar la cita.


Conclusiones del Equipo
